✅ Completed Tasks

1️⃣ User & Group Management
Created devops_user and assigned them to devops_team.

Configured Sudo privileges and secured SSH access via /etc/ssh/sshd_config.

2️⃣ File Permissions (PoLP)
Established /devops_workspace with strict 640 permissions.

Owner: Read/Write | Group: Read | Others: No Access.

3️⃣ Log Analysis (The Big Three)
Grep: Isolated "ERROR" entries.

Awk: Extracted specific timestamp columns.

Sed: Redacted sensitive IP addresses for security compliance.

4️⃣ Volume Management
Mounted external storage using Loop Devices.

Monitored disk health using df -h and du -sh.

5️⃣ Process Control
Managed background tasks using & and jobs.

Mastered system monitoring with top, htop, and kill commands.

6️⃣ Automation (The DevOps Way)
Created a Bash Script to automate directory backups.

Scheduled the script using Cron for midnight execution.

Custom Tooling: Wrote a log_filter.sh script to color-code Error and Warning logs.
